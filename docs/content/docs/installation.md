## Scaffold a new project

The recommended way to start a new v-ibe project is using the official template:

```bash
npx degit arboledavargas/v-ibe-template my-app
cd my-app
npm install
npm run dev
```

This sets up a Vite project pre-configured with TypeScript, JSX, and the v-ibe Vite plugin.
